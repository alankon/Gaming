# Sound Notes

Aprender Teclas now uses procedural WebAudio imitations instead of real animal recordings.

This keeps the game lighter, more playful, and more authorial:

- no external audio files are required during gameplay;
- each key stops the previous imitation before starting the next one;
- animal and object sounds are intentionally cute, child-friendly approximations.

The `static/sounds` folder is kept only for this note and future optional assets.
